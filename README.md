# Password Manager Web App (Flask + SQLite + Bootstrap)

Web-based Password Manager built with **Python Flask**, **SQLite**, and **Bootstrap 5 dark UI**.  
Features include saving website credentials, notes, search, delete, and CSV export.

- SQLite backend (single `.db` file, easy to backup)
